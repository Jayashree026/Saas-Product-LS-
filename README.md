# TalkToDB  

**Natural language to SQL query converter and executor**  

##  Features  

### Supported Query Types  
- `SELECT` statements  
- `JOIN` operations  
- Aggregate functions:  
  - `COUNT()`  
  - `AVG()`  
  - `SUM()`  

### Output Format  
- Returns both:  
  1. Generated SQL query  
  2. Execution results (for SELECT queries)
